# BuildKorbo?

A Bangladesh-focused business and startup validator that helps users evaluate whether an idea can realistically work in the local market.

## Overview

**BuildKorbo?** is a frontend web application that analyzes business ideas using practical Bangladesh-specific signals such as district context, university ecosystem, budget level, founder profile, and execution environment. It turns a short idea prompt into a structured validation report with scoring, risks, launch guidance, and early go-to-market suggestions.

## Features

* Business idea scoring and validation
* Bangladesh district-based market context
* University ecosystem analysis
* Founder and budget-fit evaluation
* Risk and competition assessment
* MVP and launch planning suggestions
* First-user acquisition guidance
* Local history saving with browser storage
* Chart-based score visualization
* Clean landing page and polished dashboard UI

## Core Analysis Factors

The app evaluates ideas based on:

* Market fit
* Bangladesh fit
* Monetization potential
* Feasibility
* Founder fit
* Idea clarity
* Competition level
* Trust barrier

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Visualization:** Chart.js
* **Storage:** LocalStorage
* **Architecture:** Client-side rule-based analysis engine

## How It Works

1. The user enters a business idea
2. The app detects:

   * business category
   * city or district context
   * university context
   * founder type
   * likely budget level
3. It generates:

   * overall score
   * risk level
   * local market insight
   * MVP suggestions
   * first-user strategy
   * investor-style verdict
